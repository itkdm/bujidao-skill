# Bujidao Skill

[中文](./README.md)

A curated collection of reusable AI agent skills, prompts, and workflows for personal productivity and project development.

## Overview

`Bujidao Skill` is a place to collect AI agent skills that are actually reusable, portable, and composable.

## Why This Repo

Many useful AI agent capabilities eventually become repeatable patterns, such as:

- task-specific prompt structures
- reusable analysis procedures
- project workflow conventions
- output templates for recurring scenarios

The goal of this repository is to turn those patterns into reusable skills instead of leaving them scattered across chat history, private notes, or individual projects.

## Who It Is For

This repository is a good fit if you want to:

- build a reusable library of agent capabilities for personal projects
- standardize prompts and workflows for a team
- reuse the same high-quality skills across multiple repositories
- continuously collect AI workflows that have already proven useful

## Design Principles

- Keep it simple
- Organize around the skill itself
- Prefer reusable content over project-specific detail
- Evolve gradually as the collection grows

## Repository Structure

The repository follows a flat structure:

```text
bujidao-skill/
  skill-a/
  skill-b/
  skill-c/
  README.md
  README.en.md
  LICENSE
```

Each root-level skill folder should contain the files it needs. When applicable, `SKILL.md` is recommended as the main entry point.
